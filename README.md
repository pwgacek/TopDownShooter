# TopDownShooter
python project - IV semester<br>
python version: 3.9.13
## Objective
Try to survive
## Controls
**WSAD** - moving <br>
**Mouse** - aiming <br>
**R** - reload <br>
**Scroll/Numbers** - change weapon <br>
**Left Mouse Click** - shoot <br> 
**F3** - show fps <br>
**F4** - show number of monsters <br>
